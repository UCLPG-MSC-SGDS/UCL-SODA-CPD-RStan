Introduction to Bayesian Inference and Modelling (2023/24)

1 What is Stan?

Stan is an interface for several statistical software packages (e.g., RStudio, Python, Julia, Stata, and MATLAB) which allows the user to perform state-of-the-art statistical modelling within a Bayesian framework. The focus of this workshop will be solely on Stan and RStudio. We will show you how one can develop and compile Stan scripts for Bayesian inference through RStudio to perform basic parameter estimation, as well as a wide range of regression-based techniques starting with the simplest univariable linear models and its different families to the more advanced multivariable spatial risk models.

Before all that, let us install the appropriate software. The next sections will guide you through the installation process.

1.1 Installation of R and RStudio (Window and MAC)

Make sure to install the latest version of R and RStudio that is appropriate for your operating system.

As of June 27, 2023, the latest version of RStudio (i.e., Windows 10/11 or macOS 11+) is version 2023.06.0-421. For base R, it is version 4.3.1.
